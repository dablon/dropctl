📦 dropctl

Secure peer-to-peer file transfer CLI - No cloud, no middleman

🔒 Encrypted • 🌐 P2P • 🚀 Fast
📥 Download ⭐ Star on GitHub

Why dropctl?

Direct file transfer between computers

🔒
End-to-End Encrypted
Your files are encrypted before transfer. Only you and the recipient can decrypt them.
🌐
P2P Direct Transfer
Files transfer directly between computers. No cloud storage, no data collection.
🚀
Fast Transfers
Maximum speed using your full bandwidth. No upload/download bottlenecks.
🔗
Simple Connection
Connect using peer IDs or generate a one-time code to share.
📱
Cross-Platform
Works on Linux, macOS, and Windows. Same CLI experience everywhere.
🔐
No Account Required
No registration, no login. Just install and transfer.

🚀 Quick Start

terminal 
# Install dropctl
cargo install dropctl

# Send a file
dropctl send myfile.zip

# Receive a file (share the code with sender)
dropctl receive

📦 Installation

Install with Cargo

cargo install dropctl
